/pointsofinterest?lat=?&lng=?&radius=?

Optional request header: "types"

the location point from where the search its done is defined by lat(latitude) and lng(longitude) parameters

radius parameter defines the area from which points of interests are retrieved

if types headers exists and its a CSV string, then only the points of interest that matches at least one type in the list will be returned in result
